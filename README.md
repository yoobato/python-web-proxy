# python-web-proxy
2018-2 Yonsei University / Computer Network - Project 2

### Environments
- Python 2.x

### Usage
- Naive
```sh
python proxy.py
```

- Persistent connection
```sh
python proxy.py -pc
```

- Multithreading
```sh
python proxy.py -mt
```

- Persistent connection & Multithreading (Best performance)
```sh
python proxy.py -pc -mt
```
